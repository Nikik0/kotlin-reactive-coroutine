# kotlin-reactive-coroutine

Minimalistic crud project demostraiting building service using spring webflux and coroutines.

## Description

Used technologies:
* Kotlin
* Spring boot
* Webflux
* Coroutines
* PostgreSQL (r2dbc driver)
* Flyway

## Getting Started

### Prerequisites

* Docker

### Installing

* No installing needed, just run it with docker

### Executing program

* Run the compose
```
docker-compose up -d
```
